A simple and smooth solution to find existing verified ayush centre and option to remotely connect with the verified doctor and authentic information 
Ayusutra Harmony Flow Welcome to the official repository for Ayusutra Harmony Flow. This web application allows users to discover, book, and manage Panchakarma therapies at certified Ayurveda centers across India.  Features Discover Centers: Browse and filter a curated list of top-rated Ayurveda centers.  Book Therapies: Securely book appointments for various Panchakarma treatments.  User Dashboards: Manage your bookings, view treatment history, and track your wellness journey.  Admin Panel: A dedicated dashboard for center administrators to manage bookings, schedules, and listings.  Interactive Map: Visually explore the locations of different centers.  Tech Stack Framework: React with Vite  Language: TypeScript  Styling: Tailwind CSS  Getting Started To run this project locally, follow these steps:  Install dependencies:  npm install  Run the development server:  npm run dev  Build for Production To create an optimized production build, run:  npm run build  This will generate a dist folder which can be deployed to any static site hosting service.
